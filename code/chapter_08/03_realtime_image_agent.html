You’ll be prompted for microphone access on connect.
Get a client ephemeral key (valid briefly):
# PowerShell (no jq needed)
    (Invoke-RestMethod -Uri "https://api.openai.com/v1/realtime/sessions" -Method POST `
      -Headers @{ Authorization = "Bearer $env:OPENAI_API_KEY" } `
      -ContentType "application/json" `
      -Body '{"model":"gpt-4o-realtime-preview-2025-06-03"}').client_secret.value
In production, mint the key on your backend and pass it to the browser after auth.
